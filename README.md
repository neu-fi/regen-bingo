# regen-bingo
A global trustless Bingo game where half of the proceeds go to public good funding.

## Development
Use Node 16+ and Yarn 3.

### Tab 1

#### Install dependencies
```
yarn
```

#### Run Hardhat Chain
```
yarn chain
```

### Tab 2
While keeping the `yarn chain` tab open, follow the instructions below in a second tab.

#### Run the frotnend
```
yarn dev
```

## Documentations
